--- layout: page title: History description: How it happend? ---

This project had been started as a python REST library for kaa. after project had grew, we decided to complete it as stand alone middleware.

In the begining we used SDN101 project, which is a gread memory for Parham Alvani good old days, as a good example of python REST API client.

When 18.20 was leaving us alone, we created improved version of 18.20. this version had its burial with Aolab in 2017.

Finally Bamboo is raised from the ashes of Aolab by Bambil open source organization of Iman Tabrizian who is architectural designer of Bamboo.
