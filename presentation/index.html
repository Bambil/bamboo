Major Components in Management API

Attributes (RO)
Fairly static properties of a modeled object
Statistics (RO)
Useful information about what a system has been doing
States (RO)
The current condition of a system
Settings (WO)

I1820 Plugin System

Keywords

Chain
Ordered array of plugins
Plugin
Customizable python code that runs on each log receiving
Filter Plugin
Plugin that can stop the chain if you IF is false
Notifier Plugin
Plugin that sends notification to your target device

How we chain the plugins

  • Create a chain
  • Add filters and notifiers
  • While filters return True chain continue to run
  • While chain continue to run notifers run and notify your devices
